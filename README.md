Yunpian Notifier
===============

Provides [Yunpian](https://www.yunpian.com) integration for Symfony Notifier.

DSN example
-----------

```
YUNPIAN_DSN=yunpian://APIKEY@default
```

where:
 - `APIKEY` is your Yunpian API key
